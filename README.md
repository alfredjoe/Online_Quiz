# Quiz Portal Website

An interactive web platform that allows users to create quizzes, attend quizzes, and add quiz questions by uploading PDF files.

## Features

- User authentication (signup/login).
- Role-based access for quiz creators and participants.
- Create, edit, and delete quizzes with ease.
- Upload PDF files to automatically extract and add quiz questions.
- Participate in quizzes with real-time or asynchronous modes.
- Timed quizzes with automatic submission and instant scoring.
- Detailed result analytics and leaderboards.
- Admin panel for managing users and quizzes.

## Technology Stack

- Frontend: React.js (or Angular/Vue)
- Backend: Node.js with Express (or Django/Flask)
- Database: MongoDB/PostgreSQL
- File Storage: Cloud storage (e.g., AWS S3) for PDF uploads
- PDF Parsing: Libraries like pdf-lib or PyPDF2
- Authentication: JWT/OAuth

## Getting Started

### Prerequisites

- Node.js and npm installed
- Database (MongoDB/PostgreSQL) set up
- Cloud storage credentials for file uploads
